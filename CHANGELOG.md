# Changelog

Все значимые изменения в проекте документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.5.0] - 2025-04-25

### Added

- **Главная страница магазина**
  - Слайдер баннеров с автоматическим переключением (5 секунд)
  - Секция приветствия с описанием магазина
  - Секция популярных товаров с карточками
  - Секция "Как нас найти" с картой

- **Новые UI-компоненты**
  - `BannerSlider` — слайдер изображений с навигацией и точками
  - `IntroSection` — блок приветствия на главной странице
  - `LocationSection` — секция с картой и контактной информацией
  - `PopularProductsSection` — список популярных товаров

- **Данные для баннеров**
  - Добавлен файл `banners.js` с данными для слайдера
  - 4 баннера с тематикой электроники и гаджетов

- **Утилиты**
  - Добавлена утилита для работы с массивами (`array.js`)

- **Тестирование**
  - Тесты для главной страницы
  - Тесты для BannerSlider
  - Тесты для IntroSection
  - Тесты для LocationSection
  - Тесты для PopularProductsSection
  - Тесты для утилиты массивов

### Changed

- Обновлён компонент Header для интеграции с главной страницей
- Обновлён компонент App для поддержки маршрутизации на главную страницу
- Оптимизирована CatalogPage — удалены избыточные стили

### Fixed

- Исправлены тесты компонента App после добавления главной страницы

## [1.4.0] - 2025-04-21

### Added

- **Страница детального описания товара**
  - Детальная информация о товаре (характеристики, описание)
  - Галерея изображений с навигацией
  - Отображение цен со скидками
  - Управление количеством товара на странице
  - Список похожих товаров

- **Новые UI-компоненты**
  - `ProductGallery` — галерея изображений с слайдами и точками навигации
  - `ProductInfo` — блок информации о товаре с ценами и описанием
  - `RelatedProducts` — список связанных товаров
  - `BackButton` — кнопка возврата к каталогу
  - `NotFoundState` — состояние при отсутствии товара

- **Новые хуки**
  - `useProductDetail` — управление данными страницы товара
  - `useGallery` — управление состоянием галереи
  - `useCartItem` — управление товаром в корзине

- **Улучшения каталога**
  - Отображение количества товара в карточке при наличии в корзине
  - Ссылка на детальную страницу с карточки товара

- **Тестирование**
  - Тесты для страницы детального описания товара
  - Тесты для всех новых компонентов (ProductGallery, ProductInfo, RelatedProducts и др.)
  - Тесты для новых хуков

### Fixed

- Исправлены кнопки добавления в корзину на странице детального описания

## [1.0.0] - 2025-01-17

### Added

- **Базовый функционал магазина**
  - Список товаров с возможностью добавления в корзину
  - Корзина товаров с управлением количеством
  - Система скидок на товары
  - Система промокодов для дополнительных скидок
  - Ограничение количества товаров в корзине

- **UI-компоненты**
  - Футер с контактной информацией
  - Интеграция карты (компонент Map)
  - Миниатюры товаров в корзине
  - Обработка пограничных ошибок (error boundaries)

- **Документация и инфраструктура**
  - Добавлен README с описанием проекта
  - Настроены engines для фиксации версий Node.js
  - Добавлены правила для AI-ассистента

- **Тестирование**
  - Тесты для компонента карточки товара
  - Тесты для страницы корзины
  - Тесты для остальных страниц приложения
  - Тесты для компонента App
  - Тесты для всех UI-компонентов
  - Тесты для утилит и хуков
  - Тесты для новых компонентов (футер, карта)

- **Линтеры и качество кода**
  - Настроен ESLint для JavaScript/JSX
  - Настроен Stylelint для CSS
  - Добавлена JSDoc-документация для компонентов
  - Исправлены все ошибки линтера

### Changed

- **Рефакторинг компонентов**
  - Декомпозиция страницы корзины на мелкие компоненты
  - Улучшена структура и читаемость компонентов
  - Оптимизирован компонент Map
  - Заменены изображения товаров

- **Зависимости**
  - Зафиксированы версии всех зависимостей для воспроизводимости сборки
  - Установлены зависимости для тестирования (Vitest, Testing Library)
